# MathVerse Knowledge Platform

Purpose

The Knowledge Platform is the educational foundation of MathVerse AI.

It provides structured educational knowledge to the Tutor Brain.

The Tutor Brain never reads PDFs directly.

Instead, it queries the Knowledge Platform.

The Knowledge Platform is built from:

- Official syllabus
- NCERT
- PYQs
- Coaching material
- Standard reference books
- Teacher authored content

Output

Structured educational objects.
